### Recommended Setup
1. Create a folder in your documents titled "SPM VD Apps"
2. Select the installer zip file from the sharepoint
3. Create a seperate subfolder within SPM VD Apps for each application
4. Extract the zip file into that subfolder.
5. Click the "create_shortcuts_exe.vbs" file. This will create the shortcuts on your desktop and start menu for the app.
6. Run App using the shortcuts. Having them unzipped in the same folder is crucial.
